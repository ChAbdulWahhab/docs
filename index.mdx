## 1. JavaScript Syntax & Values

### What is JavaScript?

JavaScript is a dynamic programming language used for web pages (browser), servers (Node.js), and more.

### First example (console):

```js
// Hello world
console.log('Hello, JavaScript!');
```

<Card title="Key concepts" icon="lightbulb" href="/basics/key-concepts">
  - Statements & expressions
  - Comments
  - Semicolons (optional but consistent style helps)
</Card>

### Variables

```js
let name = 'Ali';       // changeable
const pi = 3.14159;     // constant
var oldStyle = 10;      // avoid: function-scoped
```

**Exercise:** create 3 variables (string, number, boolean) and log them.

## 2. Types & Operators

### Primitive types

* `string`, `number`, `boolean`, `null`, `undefined`, `bigint`, `symbol`

```js
typeof 42;          // "number"
typeof 'hi';        // "string"
```

### Operators

* Arithmetic: `+ - * / %`
* Comparison: `== === != !== > <`
* Logical: `&& || !`

**Tip:** use `===` (strict equality) to avoid coercion surprises.

---

## 3. Control Flow

### If / else

```js
const age = 18;
if (age >= 18) {
  console.log('Adult');
} else {
  console.log('Minor');
}
```

### Loops

```js
for (let i = 0; i < 5; i++) console.log(i);
let j = 0; while (j < 3) { console.log(j); j++; }
```

**Exercise:** write a loop that prints even numbers between 0 and 20.

## 4. Functions

```js
// function declaration
function add(a, b) {
  return a + b;
}

// arrow function
const square = x => x * x;
```

## 5. Objects & Arrays

### Objects

```js
const person = {
  name: 'Sara',
  age: 25,
  greet() { console.log('Hello ' + this.name); }
};
person.greet();
```

### Arrays

```js
const arr = [1, 2, 3];
arr.push(4);
arr.map(x => x * 2);
```

## 7. Asynchronous JavaScript

### Callback

```js
setTimeout(() => console.log('Later'), 1000);
```

### Promise

```js
fetch('/data.json')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
```

### Async / Await

```js
async function load() {
  try {
    const res = await fetch('/data.json');
    const json = await res.json();
    console.log(json);
  } catch (e) {
    console.error(e);
  }
}
```

**Exercise:** write `async` function that waits 2 seconds then returns "done".
